- Validating user input**
- Working with dates (incrementing, validation, formatting, etc.)

 Features:
 Input Validation:
- Validate integers, doubles, ranges, and strings.
- Clean and reusable validation functions.
- Encapsulated inside the `clsInputValidate` class.

 Date Utilities:
- Date formatting and parsing.
- Adding days to a date.
- Comparing two dates.
- Struct-based date representation.
- All logic inside `clsDate` utilities.
